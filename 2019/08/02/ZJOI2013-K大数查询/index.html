整体二分裸题

Solution

所谓整体二分，需要数据结构题满足一下性质

  1. 询问的答案具有可二分性
  2. 修改对判定答案的贡献互相独立，修改之间互不影响效果
  3. 修改如果对判定答案有贡献，则贡献为一确定的与判定标准无关的值
  4. 贡献满足交换律、结合律，具有可加性
  5. 题目允许离线算法

论文中还提到我们在处理的过程中，复杂度绝对不能和序列总长$n$线性相关，只能和当前处理序列区间的长度相关
一个显然的推论就是我们清空辅助的数据结构 (如BIT等) 不能直接清空而要逆向地加 / 减回去
