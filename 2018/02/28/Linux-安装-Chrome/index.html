Linux 安装 Chrome

发表于 2018-02-28

题外话

貌似NOIP许多省开始不用Windows改为直接用Linux，作为Linux都不会太用的蒟蒻，还是有些担心的。
发觉elementaryOS挺漂亮的，而且基于Debian和Ubuntu，于是开始用elementaryOS。

elementaryOS

坑

首先elementaryOS的关闭键在左上角，跟macOS差不多了，而且没有最小化
其次elementaryOS单击就是打开文件，用惯Windows的我根本不习惯
最后elementaryOS没有自带rpm，这简直惨绝人寰啊

优点

当然elementaryOS还是无可厚非的
首先，它作为最漂亮的Linux系统，向macOS学习了不少界面美化的技巧
其次，它内置G++和GCC，并且持续更新（赞）

正题

当然这次主要讲的是安装chrome的经过……
内置的Epiphany浏览器还不错，就是滚动条太细了，UI也比较臃肿，于是果断换chrome

失败3

当时我简直快要热泪盈眶啊

又一个题外话

Linux下我的标准配置是VSCode+Sublime Text 3，所幸这两个软件的安装比Chrome不知道方便多少倍了
Linux真的是一个大佬们才能用的系统啊！
