「C++技巧」 计时

发表于 2017-07-31

说明

计时可以在C++里面方便地实现，可以在诸如NOIP、NOI之类的比赛中了解算法最坏运算时间。
当然，如果对于自己算法的时间复杂度很有信心的话（\(O(N)\)之类的）请移步……不过如果复杂度到了\(O(N^2)\)或\(O(N \log N)\)的级别的话还是建议测一下，前提是你会敲……毕竟第2、3种方法比较复杂，考试时用的是Dev-C++又不是Visual Studio。

方法1

用time()函数获取当前系统时间。
time()函数返回的是time_t类型的数据，返回值是从1970年1月1日（UNIX TIME的纪元时间）到当前时刻的秒数。
