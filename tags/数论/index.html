Link

Solution

$$
\begin{aligned}
X_{i + 1} &\equiv aX_i + b(\mathrm{mod},p)\notag\\
X_{i + 1} + \frac{b}{a} &\equiv aX_i + b + \frac{b}{a}(\mathrm{mod},p)\notag\\
aX_{i + 1} + b &\equiv a^2X_i + ab + b(\mathrm{mod},p)\notag\\
X_{i + 2} &\equiv a^2X_i + ab + b(\mathrm{mod},p)\notag\\
\end{aligned}
$$
